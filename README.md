# homebrew-tools

Personal Homebrew tap for custom formulae and utilities.

```bash
brew tap mythkiven/tools https://github.com/mythkiven/homebrew-tools
brew update
```

## Formulae

| Formula | Description |
|---------|-------------|
| `crashDecoder` | iOS crash log analysis and symbolication |

Install example:

```bash
brew install mythkiven/tools/crashDecoder
```

Upgrade:

```bash
brew upgrade mythkiven/tools/crashDecoder
```

Uninstall:

```bash
brew uninstall mythkiven/tools/crashDecoder
```
